## Getting Started

### Adding Questions

You can add a new question at the top level, or you can go to each section and add a question to that specific section:

![Add to Section](/images/add-to-section.png)

### Reordering Form Elements

You can click and drag rows to move their position in the form. The order the elements appear in the list is how they will be displayed in the form.

### Editing Sections

You can also edit sections:

![Edit Sections](/images/edit-sections.png)

## Data Structure

In the admin list, we have the following columns:
- **Title** - The question or label text
- **Subtitle** - Optional additional text
- **Section** - Which section the question belongs to
- **Input Type** - The type of form input
- **Dropdown Options** - Options for dropdown and likert inputs (comma-separated values)
- **Likert Options** - Options for likert scale inputs (comma-separated values)

## Input Types

### Text Input

**Required Fields:** Title, Section, Input Type

**Example Data:**

| Title | Subtitle (optional) | Section | Input Type | Dropdown Options | Likert Options |
|:------|:---------------------|:--------|:-----------|:-----------------|:---------------|
| Are there any other marketing materials you need that were not listed? | | Marketing Supplies | Text | (keep empty) | (keep empty) |

**Resulting Form Input:**

![Text Input](/images/text-input.png)

### Number Input

**Required Fields:** Title, Section, Input Type

**Example Data:**

| Title | Subtitle (optional) | Section | Input Type | Dropdown Options | Likert Options |
|:------|:---------------------|:--------|:-----------|:-----------------|:---------------|
| How many items do you need? | | Marketing Supplies | Number | (keep empty) | (keep empty) |

Same structure as text input, but accepts numeric values only.

### Date Input

**Required Fields:** Title, Section, Input Type

**Example Data:**

| Title | Subtitle (optional) | Section | Input Type | Dropdown Options | Likert Options |
|:------|:---------------------|:--------|:-----------|:-----------------|:---------------|
| What are the dates of the Norma event? | | Norma mascot request | Date | (keep empty) | (keep empty) |

**Resulting Form Input:**

![Date Input](/images/date.png)

### Dropdown Single

Single-select dropdown that allows users to choose one option.

**Required Fields:** Title, Section, Input Type, Dropdown Options

**Example Data:**

| Title | Subtitle (optional) | Section | Input Type | Dropdown Options | Likert Options |
|:------|:---------------------|:--------|:-----------|:-----------------|:---------------|
| Do you need Murdoch's Logo Stickers? | | Marketing Materials | Dropdown Single | 100,200,300 | (keep empty) |

**Resulting Form Input:**

![Dropdown Single](/images/dropdown-single.png)

### Dropdown Multiple

Multiple-select dropdown that allows users to choose multiple values.

**Required Fields:** Title, Section, Input Type, Dropdown Options

**Example Data:**

| Title | Subtitle (optional) | Section | Input Type | Dropdown Options | Likert Options |
|:------|:---------------------|:--------|:-----------|:-----------------|:---------------|
| Which marketing materials do you need? | | Marketing Materials | Dropdown Multiple | Brochures,Flyers,Posters,Banners | (keep empty) |

**Resulting Form Input:**

![Dropdown Multiple](/images/dropdown-multiple.png)

### Likert

A likert scale input that combines dropdown options with likert scale options.

**Required Fields:** Title, Section, Input Type, Dropdown Options, Likert Options

**Example Data:**

| Title | Subtitle (optional) | Section | Input Type | Dropdown Options | Likert Options |
|:------|:---------------------|:--------|:-----------|:-----------------|:---------------|
| Murdoch's Rewards - Signage | | Marketing Materials | Likert | 4,6,8 | Register Sign - English,Register Sign - Spanish,Poster - English,Poster - Spanish |

**Resulting Form Input:**

![Likert](/images/likert.png)

### Checkbox

**Required Fields:** Title, Section, Input Type

**Example Data:**

| Title | Subtitle (optional) | Section | Input Type | Dropdown Options | Likert Options |
|:------|:---------------------|:--------|:-----------|:-----------------|:---------------|
| I agree to the terms and conditions | | Marketing Supplies | Checkbox | (keep empty) | (keep empty) |

**Resulting Form Input:**

![Checkbox](/images/checkbox.png)

## Important Notes

<Warning>
  When creating dropdowns and likerts, do NOT include spaces after or before commas in the comma-separated values.
  
  **Correct Format:**
  - `"1,2,3"` (without spaces) ✅
  
  **Incorrect Format:**
  - `"1, 2, 3"` (with spaces) ❌
</Warning>
