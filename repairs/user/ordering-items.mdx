## How to Create a New Repair

Follow these steps to create a new repair order:

### Step 1: Click New Repair

To begin creating a new repair, click the "New Repair" button:

![New Repair Button](/images/repair-created.png)

### Step 2: Select a Customer

The first step in the repair form is to select a customer for the repair:

![Create Repair Form](/images/create-repair-form.png)

#### Option A: Select an Existing Customer

If the customer already exists in the system, you can select them from the dropdown:

![Select Customer](/images/select-customer.png)

#### Option B: Create a New Customer

If the customer doesn't exist, you can create a new customer record.

**Search BlackPurl Database**

You can search for a customer in your designated shop's BlackPurl database. If they exist, their data will automatically populate in the form:

![Search BlackPurl Customer](/images/search_blackpurl_customer.png)

**Manual Customer Entry**

If no customer is found in the BlackPurl database, fill out the customer fields manually. The following fields are required:
- First name
- Last name
- Phone number

Once you save the customer, you can then select them for the repair.

### Step 3: Fill Out Product Information

After selecting a customer, fill out the remaining data on the form. You must select a customer before proceeding with this step.

![Product Info Search](/images/product-info-search.png)

**Using the Lookup Feature**

You can use the new lookup feature to search BlackPurl data to find the best match for the following fields:
- Manufacturer/Make
- Equipment Type
- Model Number

**Required Fields**

The following fields are required to complete the repair:
- Manufacturer/Make
- Equipment Type
- Model Number
- VIN
- Description

**Optional Information**

You can also provide optional warranty information if applicable.

### Step 4: Complete Your Repair

Once all required fields are filled out, you're done! Your repair order has been successfully created.
