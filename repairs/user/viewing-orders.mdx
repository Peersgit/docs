## Viewing Repairs

This guide covers how to view, edit, and manage your repair orders.

## Viewing Options

You can right-click on any row in the repairs list to access the following options:
- View the repair
- Edit the repair
- Download the print-off version of the repair
- Download the invoice attachment (if available)

![Repair Dropdown Menu](/images/order-dropdown.png)

## Viewing Repair Details

When you select "View" from the context menu, you can see all the repair details:

![View Repair](/images/view-repair.png)

<Warning>
  You can only attach invoice information to a repair if you work at an approved Murdoch's shop location.
</Warning>

## Editing Repairs

You can also edit repairs directly from the context menu:

![Edit Repair](/images/edit-repair.png)

### How to Update Status

To update a repair's status:

1. Click the status dropdown menu
2. Select the new status
3. Save the edited repair

![Update Status](/images/update-status.png)

## Print-Off

The print-off version of a repair will look like the following:

![Print Off](/images/print-off.png)

You can use this print-off for delivery tracking or print the repair for any other needs you might have.
