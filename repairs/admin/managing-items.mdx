## Home Office Repair Management

Home office administrators follow the same process as basic users for creating repairs, with one additional requirement.

## Store Location Selection

Before creating a repair, administrators must select a store location. This ensures the repair is associated with the correct store:

![Location Picker](/images/location-picker.png)

After selecting the store location, you can proceed with creating the repair for that specific store location. Follow the same steps outlined in the [Creating Repairs](/repairs/user/ordering-items) guide for basic users.
