## Access Requirements

### Access Denied Messages

If you receive an "Access Denied" message when trying to access internal apps, please contact [IT Support](https://murdochs.desk365.io/app/tickets/createticket) for assistance.

### Mobile Device Restrictions

Mobile devices are restricted for certain pages. Only the <a href={`${site_url}/phones`} target="_blank" rel="noopener noreferrer">Murdoch's Phone Book</a> is available for mobile viewing.
